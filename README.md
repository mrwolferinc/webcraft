# Webcraft
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**Webcraft** is a web application that is simply just an online version of the popular game *Minecraft*. It is free, open-source, and even works offline.

## Controls
* <kbd>WSAD</kbd> &#x2014; Movement
* <kbd>Space</kbd> &#x2014; Jump
* <kbd>1-9</kbd> &#x2014; Select item from inventory
* <kbd>Esc</kbd> &#x2014; Pause
